Oregon Trail Remake

This individual project was the final assignment for CSCI1300- Intro to Computing, which I took my freshman year. It is meant to be an adventure game along the lines of the classic 'Oregon Trail' videogame, originally playable on the Apple II computer. This was my first large, open-ended project at CU and also one of my first forays in to object oriented programming. It is written in C++ and is played via the command line.

If you would like to view this project's source code or download it and play it for yourself you can do so here.

Scrabble Helper

This was a group project created as the final assignment of CS2270- Data Structures. It is a program designed to help you beat your friends at Scrabble by finding the best possible word to play. This was the first project where I collaborated with others to develop the final product. The large scale of the project along with the fact that there were multiple contributors meant that this was my first time using git for version control throughout the project. The program is written in C++ and is used via the command line.

If you would like to view this project's source code or download it and use it yourself you can do so here.

@twitter_poet

This individual project was created for T9Hacks 2020. It uses the Twython Twitter Search API wrapper package to grab tweets using a keyword from the user and then pull out snippets that make Haiku poems.
